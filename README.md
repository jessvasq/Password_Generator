# SecuKeys

SecuKeys is a mobile-focused application for securely storing and managing user passwords across multiple accounts.

The project explores secure data handling, authentication concepts, and backend-driven workflows for sensitive user information.

---

## Overview

SecuKeys allows users to generate, store, and manage passwords using a single master password.  

This project was built to practice backend security fundamentals, data storage, and API-driven application design.
